# Doctor Booking System

A modern healthcare appointment booking web application.

## Features

- Real-time doctor appointment booking
- Slot availability system
- Dashboard analytics
- Patient data storage
- Appointment confirmation PDF
- Responsive UI

## Tech Stack

Frontend
Next.js
TailwindCSS

Backend
Node.js APIs

Database
Supabase

Deployment
Vercel

## Installation

npm install
npm run dev

## Deployment

Frontend deployed using Vercel.

Database hosted on Supabase.
